## Pre-Requisites
Secrets needed 
```
secrets/
├── admin-password (to set grafana admin Password)
├── admin-user    ( To set grafana admin username)
├── grafana-env-secret  
```
## Deploy
1. To install the mon-stack
```./install.sh```
This installs the whole mon-stack - prometheus, grafana, loki and promtail.
